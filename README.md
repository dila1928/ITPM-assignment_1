Singlish → Sinhala Translator Testing

Automated testing project for evaluating a Singlish to Sinhala translation system using Python.

 Features

50 automated test cases
Covers 24 Singlish input types
Compares expected vs actual outputs

Run in Terminal:

python test_automation.py --excel "Assignment 1 - Test cases.xlsx" --url "https://www.pixelssuite.com/chat-translator"
```

Files

test_automation.py` – automation script
Assignment 1 - Test cases.xlsx` – test data
